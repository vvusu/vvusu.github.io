PVE 安装后需要做的事情

Created2022-06-23|Updated2023-09-20
|Post View:

系统更新

取消订阅

nano /etc/apt/sources.list

取消订阅
nano /etc/apt/sources.list.d/pve-enterprise.list

注释掉订阅

apt-get update

初始化磁盘

fdisk /dev/sdc

进入fdisk
输入 p 查看磁盘分区
输入 d 删除分区
输入 w 保存
在命令行中查看磁盘 S.M.A.R.T 信息： smartctl -a /dev/sdc

打开 iommu & pci passthrough

直通硬件，主板支持 iommu

Inter cpu填写 intel_iommu=on

AMD CPU填写 amd_iommu=on

填写完 使用 update-grub 更新 grub

命令行运行 dmesg | grep -e DMAR -e IOMMU 如果没有输出，则说明有问题。

更新完后 reboot 系统。
其他参数：

关闭核显，

添加 VLan 桥接网口

查看网络端口信息

硬盘 直通

GPU 直通

设置 grub
nano /etc/default/grub

update-grub

// 设置 modules
nano /etc/modules

// 重启
reboot

// 修改虚拟机配置文件 修改cpu
nano /etc/pve/qemu-server/100.conf

// 屏蔽掉系统驱动
nano /etc/modprobe.d/pve-blacklist.conf
